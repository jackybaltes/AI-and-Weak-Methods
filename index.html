Top of Wolf and Chicks Search Space

Problem Space

A problem space is the collection of all legal problem states into a tree or graph.

Connections between the nodes (problem states) is through the application of operators

Operators may be reversible which means that you can undo them and get back to the original state. But not all operators are reversible. E.g. break a cup

Problem Space

Problem space is not represented explicitly, since it is huge. Generated implicitly through the problem states, the actions (links between problem states) and the rules of the problem.

State Space Problem Solving

State Space Problem Solving search problem space for a solution

State is a complete description of the problem/world that is relevant to solving the problem

Comparison of Depth First Search (DFS) and Breadth First Search (BFS)

Depth First Search Breadth First Search
Pro: Requires only a linear amount of memory Find optimal solution
Con:

May be sidetracked down a deep subtree

Need to backtrack

Need to check for loops

 Requires an exponential amount of memory (all nodes at a level)

Guidance for Selecting a Search Algorithm

If the tree is uniform (all the solutions are at the leaf nodes, and all leaf nodes are at the same level), then DFS is a good choice.

Examples of problems with a uniform search space would be N-Queens or Sudoku. In N-Queens: must place exactly 8 queens. You must place exactly n missing numbers.

Compare that to the Wolfs and Chicks problem - we don't know how many times we must cross the river with the raft

BFS Memory and Runtime

Assume that we search a space with a branching factor of {b} and a processing speed of {s} nodes per second

Can run an AI algorithm for a long time (e.g., Crew schedule for Air NZ run for 3 weeks on the HPC at the University of Auckland)

But running BFS, we would run out of all storage in the universe very quickly
